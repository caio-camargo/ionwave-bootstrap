### Name Type

**Compound** (Ion + Wave) — Best of both worlds: meaningful + distinctive.

**Not**:
- Descriptive ("MarineMinerals") — too generic, hard to trademark
- Invented ("Quintonix") — meaningless without context
- Metaphor ("Tide") — emotional but doesn't connect to mineralization
- Founder/Origin ("Quinton") — already taken, feels dated

## Trademark Search (Pre-Registration)

### USPTO Class 005 (Dietary Supplements)

**Search Query**: "IonWave" + phonetic equivalents ("Ion Wave", "Ion-Wave")

**Status**: **NOT STARTED** — critical blocker before printing packaging

**Process**:
1. Preliminary search via [USPTO TESS](https://tmsearch.uspto.gov)
2. If clear, hire trademark attorney for full clearance search
3. File trademark application (6-12 months to approval)
4. Secure ®  registration OR use ™ (common law) until registered

[Confidence: B | Source: Trademark registration best practices for D2C brands | Date: 2026-02-10]
