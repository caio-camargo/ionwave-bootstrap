## Trademark Registration Guide

### Step 1: Trademark Search (Before Filing)

**Pre-filing search is essential** — filing an application that conflicts with an existing mark wastes $250-350 and 8+ months.

**Search Steps:**
1. **USPTO TESS Database**: Search for exact and phonetically similar marks (free) [Confidence: A | Source: USPTO.gov/trademarks | Verified: 2026-02-11]
2. **State trademark databases**: Check states where IonWave will primarily operate
3. **Common law search**: Google, social media, domain registrars — look for unregistered use
4. **Comprehensive search** (recommended): $300-500 through trademark attorney or service (e.g., Corsearch, TrademarkNow, CompuMark)

**"IonWave" Preliminary Analysis:**
- "Ion" is commonly used in supplement branding (potential descriptiveness concern)
- "Wave" is commonly used in wellness/ocean branding
- Combined "IonWave" — assess likelihood of confusion with existing marks
- [VOID — requires: Actual TESS search results and attorney analysis]

### Step 2: Select Filing Classes

**International Classification of Goods and Services (Nice Classification):**

| Class | Description | Filing Fee | IonWave Relevance |
|-------|-------------|------------|-------------------|
| Class 5 | Dietary supplements; nutritional supplements | $250 (TEAS Plus) / $350 (TEAS Standard) | PRIMARY — file first |
| Class 32 | Non-alcoholic beverages (if liquid product) | $250/$350 | If product is ready-to-drink format |
| Class 35 | Retail store services; online retail | $250/$350 | For D2C e-commerce operations |
| Class 44 | Health and wellness services | $250/$350 | If offering health consulting/content |

**Recommendation**: File Class 5 first (dietary supplements). Add Class 35 (online retail) if budget allows. Other classes can be added later. [Confidence: B | Source: USPTO Nice Classification system, supplement brand filing patterns | Verified: 2026-02-11]

### Step 3: File Application

**Filing Options:**
- **TEAS Plus** ($250/class): Lower cost, must select pre-approved description of goods/services from USPTO ID Manual. Recommended if your goods fit existing descriptions. [Confidence: A | Source: USPTO fee schedule effective 2025 | Verified: 2026-02-11]
- **TEAS Standard** ($350/class): Higher cost, allows custom description. Use if pre-approved descriptions don't accurately describe your goods.

**Filing Basis:**
- **Use in Commerce (1(a))**: If already selling products with the mark
- **Intent to Use (1(b))**: If not yet selling but plan to. Must file Statement of Use within 6 months of Notice of Allowance (extensions available up to 3 years, $125 each). IonWave will likely file 1(b) initially. [Confidence: B | Source: 15 USC Section 1051 | Verified: 2026-02-11]

### Step 4: Examination Timeline

| Stage | Timeline | Notes |
|-------|----------|-------|
| Filing | Day 0 | Serial number assigned immediately |
| Examination | 3-6 months | USPTO examining attorney reviews |
| Office Action (if any) | +3 months response deadline | Address any objections |
| Publication | 30 days after approval | Published for opposition |
| Opposition period | 30 days | Third parties can oppose |
| Registration (1(a)) or NOA (1(b)) | After opposition period | |
| Statement of Use (1(b) only) | Within 6 months of NOA | Must show actual commercial use |
| **Total timeline** | **8-14 months** | Longer if Office Actions or oppositions |

### Step 5: Maintain Registration

- **Declaration of Use (Section 8)**: Due between 5th and 6th year after registration, then every 10 years [Confidence: A | Source: 15 USC Section 1058 | Verified: 2026-02-11]
- **Renewal (Section 9)**: Every 10 years
- **Incontestability (Section 15)**: Available after 5 years of continuous use — makes mark much harder to challenge

## IP Assignment and Protection

### Founder IP Assignment

**Every founder must execute a Technology Assignment Agreement** assigning to the company:
- All inventions, ideas, and improvements related to company business
- All prior IP contributed to the company
- All future IP created during engagement
- Moral rights waiver (to extent permitted by law)

**Critical timing**: Execute BEFORE or simultaneously with stock issuance. IP gaps are deal-breakers in due diligence. [Confidence: A | Source: Universal VC due diligence requirement | Verified: 2026-02-11]

### Contractor/Employee IP Assignment

All contractors and employees must sign IP assignment agreements that include:
- Work-for-hire provisions (for copyrightable works)
- Invention assignment (for patentable innovations)
- Confidentiality/NDA provisions
- Non-solicitation (of employees and customers)

**Note**: California law restricts non-compete agreements — employees cannot be bound by non-competes. Contractors have limited restrictions. [Confidence: A | Source: California Business and Professions Code Section 16600 | Verified: 2026-02-11]

### Trade Secret Protection

Marine plasma formulation, supplier relationships, and customer data strategies constitute trade secrets deserving protection:

| Protection Measure | Implementation | Priority |
|-------------------|----------------|----------|
| NDAs with all parties | Standard NDA template, signed before any disclosure | P0 |
| Access controls | Limit formulation details to need-to-know basis | P0 |
| Documentation | Document what constitutes trade secrets | P1 |
| Employee/contractor training | Confidentiality obligations training | P1 |
| Physical/digital security | Encrypted storage, access logs, two-factor auth | P1 |
| Exit protocols | Return of confidential materials at termination | P1 |

## Holding Company IP Strategy (Studio 4 Model)

### IP Licensing Architecture (When Applicable)

If Studio 4 Ventures becomes the holding company (see Tab 2):

```
Studio 4 Ventures (Holding Co) — OWNS all IP
  │
  ├── Licenses IP to IonWave, Inc. (operating co)
  │   └── Exclusive license for dietary supplements
  │   └── Royalty or cost-plus basis
  │
  ├── Licenses IP to Future Brand 2, Inc.
  │   └── Exclusive license for [different product category]
  │
  └── Licenses community IP to IonWave DAO LLC
      └── Non-exclusive license for community governance use
```

**Benefits of centralized IP ownership:**
- Protects IP if any operating entity faces litigation or bankruptcy
- Enables licensing to multiple entities without conflicts
- Simplifies IP due diligence in acquisitions (one entity owns everything)
- Standard in LVMH-style multi-brand structures [Confidence: C | Source: LVMH model analysis, multi-brand holding company practices | Verified: 2026-02-11]

**Costs/Risks:**
- Transfer pricing rules apply (IRS Section 482) — licenses must be at arm's length
- Additional entity complexity
- Not worth implementing until Studio 4 has 2+ operating companies
