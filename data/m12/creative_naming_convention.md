## 1. Why Naming Conventions Matter

### 1.1 The Problem

**Without Standard Naming**:
- "final_v3_FINAL_for_real.mp4" → impossible to track performance
- Team members can't find assets
- Can't identify which angle/hook/format performed best
- Swipe file becomes unusable chaos

**With Standard Naming**:
- "IW_Meta_MissingMineral_UGC_HookA_15s_v1.mp4" → instantly know: platform, angle, format, hook, length
- Easy to search, sort, analyze
- Performance patterns emerge ("UGC always beats founder-led on TikTok")

## 2. The Master Naming Formula

### 2.1 File Naming Structure

**Format**: `[Brand]_[Platform]_[Angle]_[Format]_[Hook/Variant]_[Length]_[Version].ext`

**Example**: `IW_Meta_MissingMineral_UGC_HookA_15s_v1.mp4`

**Breakdown**:
- **Brand**: `IW` (IonWave)
- **Platform**: `Meta`, `TikTok`, `YouTube`, `Multi` (works across platforms)
- **Angle**: `MissingMineral`, `Contrarian`, `Comparison`, `ProblemReframe`, `Authority`
- **Format**: `UGC`, `Founder`, `Static`, `AnimExplainer`, `Testimonial`
- **Hook/Variant**: `HookA`, `HookB`, `HookC`, or `V1`, `V2`, `V3`
- **Length**: `15s`, `30s`, `60s`, `90s`
- **Version**: `v1`, `v2`, `v3` (for iterations/edits)

[Confidence: A | Source: Industry standard creative file naming | Date: 2026-02]

---

### 2.2 Ad Campaign Naming Structure (Meta/TikTok/Google)

**Format**: `[Objective]_[Angle]_[Audience]_[DateLaunched]`

**Example**: `CONV_MissingMineral_Broad_0210`

**Breakdown**:
- **Objective**: `CONV` (Conversions), `TRAFFIC` (Traffic), `VIEWS` (Video Views)
- **Angle**: `MissingMineral`, `Contrarian`, `Comparison`, etc.
- **Audience**: `Broad`, `LAL1`, `Retarget`, `Interest_Health`
- **DateLaunched**: `MMDD` (e.g., `0210` = Feb 10)

**Why Date?** Easy to track campaign age and compare performance over time.

[Confidence: A | Source: M13 account_structure.md, Meta naming best practices | Date: 2026-02]

---

### 2.3 Ad Set Naming Structure (Meta)

**Format**: `[Audience]_[Placement]_[Budget]`

**Example**: `Broad_AutoPlacements_$50`

**Breakdown**:
- **Audience**: `Broad`, `LAL1-3`, `Retarget_7d`, `Interest_Biohack`
- **Placement**: `AutoPlacements`, `FeedOnly`, `Stories`, `Reels`
- **Budget**: `$50`, `$100` (daily budget)

---

### 2.4 Individual Ad Naming Structure (Meta)

**Format**: `[Angle]_[Format]_[Hook]_[Length]`

**Example**: `MissingMineral_UGC_HookA_15s`

**Why Simplified?** Ad-level name shows up in reporting. Keep it scannable.

## 5. Hook/Variant Naming

### 5.1 Hook Variants

When testing multiple hooks within the same angle, use letter codes:

- **HookA**: First hook variant
- **HookB**: Second hook variant
- **HookC**: Third hook variant
- ... continue through alphabet

**Example**:
- `IW_Meta_MissingMineral_UGC_HookA_15s_v1.mp4` → "Your electrolytes are missing 75 minerals"
- `IW_Meta_MissingMineral_UGC_HookB_15s_v1.mp4` → "I found the one ingredient LMNT leaves out"
- `IW_Meta_MissingMineral_UGC_HookC_15s_v1.mp4` → "The trace mineral complex nobody tells you about"

### 5.2 Derivative Variants

When creating derivatives of a winning ad (Tier 1 refreshes), use version numbers:

- **v1**: Original
- **v2**: First derivative (e.g., new text overlay)
- **v3**: Second derivative (e.g., new CTA)
- **v4**: Third derivative (e.g., cut-down to 15s)

**Example**:
- `IW_Meta_MissingMineral_UGC_HookA_30s_v1.mp4` → Original winner
- `IW_Meta_MissingMineral_UGC_HookA_30s_v2.mp4` → Same video, new text overlay
- `IW_Meta_MissingMineral_UGC_HookA_15s_v3.mp4` → Cut-down to 15s

## 6. Platform-Specific Naming

### 6.1 Multi-Platform vs Single-Platform

**Multi-Platform** (works on Meta, TikTok, YouTube):
- Use `Multi` in platform field
- Example: `IW_Multi_Contrarian_UGC_HookA_30s_v1.mp4`

**Single-Platform** (optimized for one platform):
- Use specific platform code: `Meta`, `TikTok`, `YouTube`
- Example: `IW_TikTok_MorningRoutine_UGC_HookA_15s_v1.mp4`

**Why Distinguish?** TikTok creative rarely works on YouTube (pacing, tone, style differ). Naming helps avoid cross-posting low-performers.

  /Assets/
    /Thumbnails/
    /Text_Overlays/

## 8. Tracking Integration

### 8.1 Creative Tracker Spreadsheet

**Columns** (align with naming convention):
- **File Name** (e.g., `IW_Meta_MissingMineral_UGC_HookA_15s_v1.mp4`)
- **Platform** (Meta, TikTok, YouTube)
- **Angle** (MissingMineral, Contrarian, etc.)
- **Format** (UGC, Founder, Static)
- **Hook/Variant** (HookA, HookB, v1, v2)
- **Length** (15s, 30s, 60s)
- **Date Launched** (YYYY-MM-DD)
- **Status** (Testing, Winner, Fatigued, Retired)
- **Spend** ($)
- **CPA** ($)
- **Hook Rate** (%)
- **ROAS** (X:1)
- **Notes** (e.g., "Winner for 3 weeks, then fatigued")

**Purpose**: One source of truth for creative performance. Easy to filter: "Show me all UGC ads on Meta with CPA <$30."

[Confidence: A | Source: M13 ad_swipe_file.md tracking structure | Date: 2026-02]

### 8.2 Swipe File Integration

When an ad becomes a winner, log it in `data/m13/ad_swipe_file.md` with:
- **File Name** (naming convention)
- **Performance Stats** (CPA, ROAS, Hook Rate)
- **Why It Worked** (angle resonance, hook strength, format fit)
- **Derivative Ideas** (how to iterate)

**Example Entry**:
```
## Winner #1: IW_Meta_MissingMineral_UGC_HookA_15s_v1

**Performance**: CPA $24, ROAS 4.2:1, Hook Rate 32%
**Date**: 2026-02-15 to 2026-03-10 (3.5 weeks)
**Why It Worked**: Curiosity gap hook ("missing 75 minerals") + UGC authenticity + fast pace (15s)
**Derivatives Tested**: v2 (new text overlay, CPA $26), v3 (cut to 10s, hook rate dropped to 24%)
**Retirement Reason**: Hook rate declined to 18%, CPM rose 40% (fatigue)
```

## 11. Version Control for Edits

### 11.1 When to Increment Version Number

**v1 → v2**: Any substantive change (new hook, new text overlay, new CTA, new cut-down)

**Do NOT increment version for**:
- Export format change (MP4 → MOV)
- Compression adjustment
- File size optimization
- Platform-specific aspect ratio export (if content is identical)

**Why?** Version numbers track creative iteration, not technical tweaks.

---

### 11.2 Version Notes

Keep a `VERSION_NOTES.txt` file in each angle folder:

**Example**:
```
IW_Meta_MissingMineral_UGC_HookA_15s_v1.mp4
- Original version
- Hook: "Your electrolytes are missing 75 minerals"
- CTA: "Try IonWave"

IW_Meta_MissingMineral_UGC_HookA_15s_v2.mp4
- Same as v1, new text overlay: "78 minerals vs 3"
- CTA unchanged

IW_Meta_MissingMineral_UGC_HookA_15s_v3.mp4
- Cut-down from 30s to 15s (removed mid-section)
- CTA: "Link in bio"
```
